Bootstrap

Making a Website With Bootstrap

Honestly it is easy using bootstrap and making small changes with css. But I definitely wont be using this ever again. It is too easy to use and somehow I feel like lacking creativity since everything is already premade? I have yet to master this (bootstrap) yet but somehow I get the feeling that I will never be able to

However I did note some important parts for the future (maybe?) usage. I have to keep in mind that the style sheet has to be copied and pasted AND the script.

You can change the order of the HTML elements but you have to keep in mind that you cannot change the class and id. If you want to change the style of a certain element, just add a style inside the html and your good to go.
