The Basic Challenges

Before starting with the intirmidate challenges of Javascript, I thought that it was better if I start with the basic challenges first. After all I have almost no experience in Javascript and have no idea where to start. I made six basic challenges. The challenges were mostly about manipulating strings and arrays.

This page is split into six sections:

  • The First Challenge | Simple Adding
  • The Second Challenge | First Factorial
  • The Third Challenge | Letter Capatilize
  • The Fourth Challenge | The Longest Word
  • The Fifth Challenge | First Reverse
  • The Sixth Challenge | Letter Change

    • The First Challenge | Simple Adding

    This just takes the parameter from the html and makes it .value. then it will calculate the simple adding and then it will alert us the result.

    The Second Challenge | First Factorial

    As the first challenge, it will take the value of the parameter from the html. Then I will make an for statement and calculate the first factorial. If it is calculated or not, the function will end by the return and alert us the new calculated factorial.

    The Third Challenge | Letter Capatilize

    This one will make the string to low letters and separate them to make it an array using the .split() function. then it will check the string and capitalize the first letter, or the first element of the array. After that it will return and join the array into a string using the array.join() function.

    The Fourth Challenge | The Longest Word

    First i will split the string using the array.split(). Then it will check the array length using the array[]1.length.

    The Fifth Challenge | First Reverse

    Again, making the string into an array using the .split() and then I use the array.reverse() to reverse the array. And then I join the array together into a string with .join().

    The Sixth Challenge | Letter Change

    I had to make an for elseif loop. This is because it will check the input each time, not only once. It will check the first condition, then second and lastly the third. If it is a letter z it will change into z. second one will check if the string has an a or z and if it does it will go to the next letter to the alphabet so a goes to z and z goes to a. the last condition will check if the letter has an a, e, I, o or an u and if it does it will make the letter into uppercase. The empty variable(result) will become the new variable, letters.
