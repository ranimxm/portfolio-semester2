Dummy JSON fetch

For this challenge I fetched a dummy JSON file and displayed the data inside the html and made a search function. I didnt really check out the css styling, since this challenge mostly was about fetching the dummy json and display it inside the html.

You can see the live demo here.

Making it fetch

First I fetched the gummyjson the way it was explained on the site itself. Afterwards I put the fetch inside the contentloaded function. This is to display the data whenever the window is loaded. The window.classmate is a global variable which holds the content of the fetched data.users.

After that I made a function displayclassmate. Inside this I put the global variable inside the html id list. Then I made a search function where the input date will be converted into lowercase and it will search the firstname and the email of the data. When searching it will clear the list whenever the search function is being used or typed in. Next I make a new li element for each user inside the data I fetched. After that I use the document.appendchild.. inside the loop, which reads each user in the filtered data list(aka classmate list).

Outside the function I wrote down an addeventlistener input for the search function. This will call the displayclassmate() function whenever the user types something inside the search bar.
